In css, which determines style - a more specific selector, or style defintion closer to the element?
It seems that a more specific selector takes precedence.
This paragraph is contained in a div with an inline color of turquoise.
However, if the style of p is defined, even at the top of the document (not inline), it takes precedence - the color of this writing is green.
Similarly, if you wish to remove text-decoration from "a" within "li", then it doesn't help to define text-decoration in "li", as somewhere higher up in the css sheets pile (perhaps default browser settings), "a" specifically is defined to be underlined.
